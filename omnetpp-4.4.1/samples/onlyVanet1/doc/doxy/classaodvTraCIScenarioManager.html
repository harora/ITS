Detailed Description

Creates and moves nodes controlled by a aodvTraCI server.

If the server is a SUMO road traffic simulation, you can use the aodvTraCIScenarioManagerLaunchd module and sumo-launchd.py script instead.

All nodes created thus must have a aodvTraCIMobility submodule.

See the Veins website for a tutorial, documentation, and publications .

Author
Christoph Sommer, David Eckhoff, Falko Dressler, Zheng Yao, Tobias Mayer, Alvaro Torres Cortes, Luca Bedogni
