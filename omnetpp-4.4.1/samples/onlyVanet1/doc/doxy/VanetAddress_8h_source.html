122 /*
123  * Stores an IPv4, IPv6 or an MACAddress address with prefix. This class should be used in
124  * vanetrouting, to guarantee IPv4/IPv6/MACAddress transparency.
125  *
126  * Storage is efficient: an object occupies size of an IPv6 address
127  * (128 bits=16 bytes) plus two short int (address type and prefix length).
128  */
